```
python3 -m venv venv       
```

para activar el entorno:
```
source ./venv/bin/activate     
```
instalar dependencias del proyecto por parte de back:
```
Pip install  flaskflask-cors
```

```
npm install
```
lanzar la aplicacion, por parte del back:
```
flask run
```
del front:
```
npm start
```
